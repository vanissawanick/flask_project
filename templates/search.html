Giphy API Search

Quick example of the Giphy Api which is pretty cool if you ask me. Just type in a search phase and get a random related GIF.

I also decided to go nuts and use ES6 features (like let, const, fat arrow function syntax and string templates) and the Traceur Compiler just to play around with whats coming up in the next generation of javascript, some pretty cool stuff.
